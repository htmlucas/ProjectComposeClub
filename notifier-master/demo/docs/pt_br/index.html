Notifier

Lindas notificações, com CSS e Javascript puro!

Trabalhando em um projeto e precisando de uma solução para notificações? Boa notícia \o/
O notifier pode te ajudar com essa tarefa! Basta importar para o seu projeto e começar a usar!

Download ZIP Projeto GitHub

Como Usar?

Muito Simples!

  1. Baixe o projeto.
  2. Exporte os arquivos.
  3. Copie os arquivos "notifier.min.css" e "notifier.min.js" para dentro do seu projeto.
  4. Importe o CSS no <header>
  5. Importe o JS antes do fechamento do <body>
  6. Agora é só ser feliz! ;)

Usando o método hide()

O método notification.show(); sempre retorna o ID da notificação que foi aberta. Nós podemos armazenar esse ID em uma variável para chamar o método notification.hide() passando esse ID para fecharmos a notificação de qualquer lugar que desejarmos.
Veja o exemplo: 

        
          var notificationId;

          var showNotification = function () {
            notificationId = notifier.show('Lembrete!', 'Você tem uma reunião agendada às 10:30h.', '', 'img/survey-48.png', 4000);
          };

          var hideNotification = function () {
            notifier.hide(notificationId);
          };

          document.querySelector('#btn-nt-show').addEventListener('click', showNotification);
          document.querySelector('#btn-nt-hide').addEventListener('click', hideNotification);

Métodos

notifier.show(title, msg, type, icon, timeout);

Parâmetro Descrição
title Título da notifiação.
msg Mensagem da notificação.
type Cor da notifiação.
Valores aceitos: '', info, success, warning, danger
icon Caminho para uma imagem que será exibida dentro da notificação.
Por Exemplo: 'img/icon.png'
timeout Tempo em milisegundos para o fechamento automático da notificação.
Só funciona se o valor passado for maior do que zero
Retorno: Id da notificação.

notifier.hide(notificationId);
Você pode usar esse método para fechar a notificação via Javascript

Parâmetro Descrição
notificationId Id da notificação
Retorno:
  • true - Se a notificação foi fechada
  • false - Se a notificação não foi fechada
